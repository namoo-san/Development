# 🔧 Development

My macOS environment setup script 👀

- macOS 開発環境
- Visual Studio Code 拡張機能
- 日常利用のアプリケーション

## 📦 Run.sh

### 🚚 Full environments setup

``` bash
sh Run.sh
```

### 🚗 Light environments setup (beta)

``` bash
sh Run-Light.sh
```
